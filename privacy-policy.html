Privacy Policy

YouTube Playlist Analyzer

This extension is designed to analyze YouTube playlist duration. Your privacy is important.

What Data Is Used

  • Playlist data requested from the YouTube Data API (playlist metadata and video durations).
  • Local cache data stored in your browser using chrome.storage.local.

What Is Not Collected

  • No personal profile is created.
  • No analytics or behavior tracking is included.
  • No data is sold or shared with third parties.

Where Data Is Stored

Cached playlist analysis is stored locally on your device in browser extension storage. The extension does not operate a remote database.

External Services

The extension communicates only with the YouTube Data API to fetch requested playlist information.
